API documentation¶

Note

This software adheres to Semantic Versioning. The current release is a beta version, hence the API is not entirely fixed yet. This means that with the deployment of new releases you should expect minor changes to existing features including additions and deletions. Once version 1.0.0 is reached the public API will be fixed.

This API documentation gives detailed information on all public functions, classes, objects, modules and subpackages provided by this package to its users. It is semi-automatically generated from source code and docstrings with the popular tool Sphinx and its autodoc and napoleon extensions. The same information can be obtained as help page with Python’s built-in help system but only in plain text instead of nicely formatted HTML documents.
