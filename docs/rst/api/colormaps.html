The following list is sorted in a hierarchy with two levels:

  1. by source of the colormap: built-in and external ones
  2. by alphabetical order of the colormap names

Note that it is currently not considered whether a colormap is sequential, diverging or categorical, which may lead to a visual impression of disorder.
