Figure¶

class Figure(html_template)[source]¶

Data structure for wrapping, displaying and exporting a JavaScript figure.

display(inline=False)[source]¶

Display the plot in a webbrowser or as IPython rich display representation.

Parameters:in_jupyter (bool) – If True, the plot will be shown inline in a Jupyter notebook.
html_text¶

Create a HTML text representation.

html_text_standalone¶

Create a standalone HTML text representation that has all javascript code embedded.

html_text_partial¶

Create a dependent HTML text representation that loads javascript with require.js.

export_html(filepath)[source]¶

Export the plot as HTML file.

Parameters:filepath (str) – Filepath of the created HTML file. If the file exists it will be overwritten without warning. If the path does not end with “.html” it will be changed to do so. If the parent directory does not exist it will be created.
Returns:filepath (str) – Filepath of the generated HTML file, guaranteed to end with “.html”.
