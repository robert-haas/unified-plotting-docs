Named colors¶

The current list of named colors stems from Matplotlib’s colors module, specifically the global variables BASE_COLORS, CSS4_COLORS, TABLEAU_COLORS and XKCD_COLORS. These names are made available both for Matplotlib and Plotly figures.

  • c (#00BFBF)

  • g (#008000)

  • k (#000000)

  • m (#BF00BF)

  • r (#FF0000)

  • w (#FFFFFF)
