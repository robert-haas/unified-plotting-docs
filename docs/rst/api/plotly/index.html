This is the subpackage unified_plotting.plotly.

It provides interactive plots based on Plotly. Its main purpose is to enable easier access to the rich plotting capabilities of this Python library by providing a simple, unified, high-level API for many plot types.

It contains the following modules and plotting functions.

  • 2D plots
    • bar
    • density_2d
    • density_scatter_histogram_2d
    • histogram_2d
    • scatter
  • 3D plots
    • contour
    • heatmap
    • scatter_3d
    • surface
  • n-D plots
    • band
    • box
    • density
    • histogram
    • parallel_coordinates
    • scatter_matrix
    • violin
  • Financial plots
    • candlestick
    • ohlc
  • Figure
