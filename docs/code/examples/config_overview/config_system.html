Configuration system¶

This Jupyter notebook provides an example of how to use the config system in unified-plotting. This feature allows to change global settings that affect each plot type. It is also possible to define styles that can be exported, imported and shared with others.

[1]:

import os

import unified_plotting as up

[2]:

x = list(range(20))
y = [item**0.5 for item in x]

Load and export defaults¶

[3]:

up.config.load_defaults()  # relevant only after some settings were changed

[4]:

data_directory = os.path.join('..', 'data', 'styles')
filepath = os.path.join(data_directory, 'exported_default_style.json')

up.config.save_defaults_to_json(filepath)

[4]:

'../data/styles/exported_default_style.json'

[5]:

up.matplotlib.scatter(x, y, x_title='Time [s]', y_title='Velocity [m/s]')

[5]:

Change current settings¶

This example shows some of the available settings.

[6]:

up.config.settings.color = ['black', 'cyan', 'magenta']
up.config.settings.paper_background_color = '#eeeeee'
up.config.settings.plot_background_color = '#aaddaa'

up.config.settings.width_mm=150
up.config.settings.height_mm=150

up.config.settings.show_title = True
up.config.settings.title = 'Kinematics'
up.config.settings.title_color = 'darkgreen'
up.config.settings.title_size = 16

up.config.settings.x_label_color = 'red'
up.config.settings.x_label_fontsize = 10
up.config.settings.x_tick_color = 'red'
up.config.settings.x_title_color = 'darkgreen'
up.config.settings.x_title_fontsize = 14

up.config.settings.y_label_color = 'red'
up.config.settings.y_label_fontsize = 10
up.config.settings.y_tick_color = 'red'
up.config.settings.y_title_color = 'darkgreen'
up.config.settings.y_title_fontsize = 14

up.config.settings.show_x_grid = True
up.config.settings.x_grid_color = '#eeeeee'
up.config.settings.show_y_grid = True
up.config.settings.y_grid_color = '#eeeeee'

up.config.settings.show_line = True

[7]:

up.matplotlib.scatter(x, y, x_title='Time [s]', y_title='Velocity [m/s]')

[7]:

Export current settings¶

Save the current style as JSON file.

[8]:

filepath = os.path.join(data_directory, 'exported_user_style.json')
up.config.save_to_json(filepath)

[8]:

'../data/styles/exported_user_style.json'

Load settings from JSON file¶

The JSON file contains a style specification that you saved for later use or that was given to you by someone else.

[9]:

filepath = os.path.join(data_directory, 'third_party_style.json')
up.config.load_from_json(filepath)

[10]:

up.matplotlib.scatter(x, y, x_title='Time [s]', y_title='Velocity [m/s]')
